## 📋 Gumloop Scraper Configuration

### Step 1: Configure the HTTP Node in Gumloop

**In your Gumloop workflow, add an HTTP Request node with these settings:**

| Setting | Value |
|---------|-------|
| **URL** | `https://master-hustle-engine.onrender.com/webhook/lead` |
| **Method** | `POST` |
| **Content-Type** | `application/json` |

### Step 2: Set the Request Body

Copy this exactly and paste into the request body:

```json
{
  "company_name": "{{company_name}}",
  "contact_email": "{{email}}",
  "website": "{{website}}",
  "industry": "{{industry}}"
}
```

**Important:** The exact field names matter — make sure your Gumloop scraper output matches:
- `company_name` (not company, not name)
- `email` (or `contact_email` — backend accepts both)
- `website`
- `industry`

### Step 3: Configure Scraper Targeting

```
Founder, Co-Founder, CEO, Owner, Managing Director,
```
